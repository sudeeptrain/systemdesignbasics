Race Condition — Two Waiters, One Order Pad

Two waiters update the same table's tally. Each reads "3 drinks", adds one, and writes "4" — but if they both read 3 at the same moment, they both write 4. Two drinks were added; the tally only rose by one. A race condition is exactly this kind of clobbered update.
