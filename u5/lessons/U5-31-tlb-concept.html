TLB (Translation Lookaside Buffer)

Paging means every memory access needs a page-table lookup — which is itself a memory access. The TLB is a tiny, ultra-fast cache of recent page→frame translations. On a TLB hit the address is resolved instantly; on a miss the OS walks the page table, then caches the result.
