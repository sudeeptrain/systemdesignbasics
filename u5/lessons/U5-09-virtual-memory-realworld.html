Virtual Memory — A Small Desk, a Big Filing Cabinet

Your desk only fits a few open folders, but the filing cabinet behind you holds thousands. You work as if you had unlimited space: only the folders you're using sit on the desk; the rest stay filed away and are fetched when needed. That's virtual memory — desk is RAM, cabinet is disk.
