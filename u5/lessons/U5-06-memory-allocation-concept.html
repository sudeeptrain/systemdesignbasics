Memory Allocation

The OS hands out chunks of memory to programs and reclaims them when freed. A simple scheme places each request in the first gap big enough (first-fit). Over time, freeing blocks in the middle leaves scattered holes — fragmentation — so free memory exists but not in one usable piece.
