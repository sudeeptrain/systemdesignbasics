Cache Hierarchy (L1 / L2 / L3 / RAM)

A CPU has layers of memory: tiny, blazing-fast L1, then larger-but-slower L2 and L3, then big-but-slow RAM. The CPU checks each level in turn; the closer it finds the data, the faster the access. A hit found deep gets pulled up to L1 for next time.
