Cache Warming & Negative Caching

Cache warming pre-loads hot data before real traffic arrives, so the first users get hits instead of a cold-start miss storm. Negative caching stores 'not found' results too — so repeated lookups for a missing key don't keep pounding the database. Both turn predictable misses into cheap hits.
