Memory Allocation — A Parking Lot

A car park hands out spaces and reclaims them when cars leave. A bus needs several adjacent spaces, so the attendant parks it in the first run of empties big enough. As cars come and go, the free spaces scatter into small gaps — so the lot may have room overall, yet nowhere to fit a bus.
