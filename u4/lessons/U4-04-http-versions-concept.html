HTTP/1 vs HTTP/2 vs HTTP/3

HTTP/1.1 sends one request at a time per connection, so a slow resource blocks the rest. HTTP/2 multiplexes many streams over one connection. HTTP/3 runs over QUIC (UDP), so a single lost packet no longer stalls every other stream.
