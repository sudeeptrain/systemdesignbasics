Web Security: XSS, CSRF, SQLi

Three classic web attacks, all from trusting input: XSS injects malicious scripts into pages, CSRF tricks your browser into firing authenticated requests, and SQL injection smuggles commands into a query. Each has a standard defense — and the rule is always: never trust input.
