API Versioning — A New Edition of a Textbook

When a textbook gets a new edition, the old one doesn't vanish — classes mid-course keep using the 1st edition while new classes adopt the 2nd. The publisher keeps both in print, nudges everyone to switch, and only retires the old edition once nobody needs it.
