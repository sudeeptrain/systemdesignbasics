Object / Blob Storage (S3-style)

Databases are wrong for big files. Object storage like S3 keeps each file as an object — its bytes plus metadata — addressed by a key in a flat bucket. It scales almost infinitely, costs little, and serves files directly, but it's not for transactions or queries.
