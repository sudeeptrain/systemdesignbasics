Inverted Index — Recipes by Ingredient

You've got eggs to use up. Reading every recipe to find which ones use eggs is slow. A cookbook's ingredient index flips it around: look up "egg" and it lists exactly which recipes contain it. That flip — from recipe→ingredients to ingredient→recipes — is an inverted index.
